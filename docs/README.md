
# Documentation folders

The folders and files for this folder are as follows:

- 'ProblemStatementAndGoals' - Contains the problem statement and goals for the project.
- 'DevelopmentPlan' - Contains the development plan for the project, including timelines and milestones.
- 'SRS' - Contains the Software Requirements Specification for the project, detailing the functional and non-functional requirements.
- 'HazardAnalysis' - Contains the hazard analysis for the project, identifying potential risks and mitigation strategies.
- 'Design' - Contains the design documentation for the project.
- 'VnVPlan' - Contains the Verification and Validation plan for the project, outlining the testing strategy and criteria for success.
- 'VnVReport' - Contains the Verification and Validation report for the project, summarizing the results of testing and validation activities.
- 'projMngmnt' - Contains project management documents, including team contribution reports.
- 'Extras' - Contains a usability report and a link to an instructional video for using the arena.
